# Go Web API Playground

A simple project created to study web API development using Go (Golang).

The purpose of this repository is not to build a production-ready application, but to serve as a sandbox for experimenting and learning important backend concepts in the Go ecosystem using stdlib as possible.

---

## Goals

This project is intended for practicing:

- Building REST APIs
- Project structure organization in Go
- HTTP routing
- Middleware
- JSON serialization
- Database integration
- Error handling
- Authentication
- Testing
- Backend development best practices
- And more...

---

## Technologies

This project currently uses or may include:

- net/http
- PostgreSQL
